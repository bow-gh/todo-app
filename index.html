Connect Firebase to share with your EA

One-time setup (~3 minutes). Once done, you and your EA open the same file and edits sync live.

1
Go to console.firebase.google.com → click Add project → give it any name → continue through the steps.
2
In your new project, click Build → Realtime Database → Create database → choose Start in test mode → Enable.
3
Go to Project Settings (gear icon) → Your apps → Add app → Web (</>) → register it. Copy the firebaseConfig object shown.
4
Paste your config JSON below and click Save & Connect.
5
Send your EA this HTML file (or host it on netlify.com/drop — just drag the file). They open it and edits sync in real-time.
